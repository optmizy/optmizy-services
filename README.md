# Optmizy — Digital Innovation Studio

A modern, interactive agency landing page built with **React 19 + TypeScript + Vite + Tailwind CSS v4**.

## ✨ What's New (v2)

| Feature | Details |
|---|---|
| **React + TypeScript** | Migrated from plain HTML to React 19 + TS + Vite |
| **Framer Motion Animations** | Smooth `whileInView`, `whileHover`, entrance animations throughout |
| **Animated Canvas Background** | Particle network with connecting lines (GPU-friendly) |
| **Contact Drawer** | Slide-in panel with full form (Name, Email, Service, Message) |
| **Floating Action Button** | Fixed bottom-right chat button opens the contact drawer |
| **Per-Service Inquiry** | "Inquire Now" appears on card hover, pre-selects service in form |
| **Custom Select Dropdown** | Animated dropdown with active state checkmarks |
| **Noise Texture Overlay** | Subtle grain for visual depth |

## 🚀 Getting Started

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # Production build → dist/
npm run preview    # Preview production build
```

## 🏗️ Stack

- **React 19** — UI framework
- **TypeScript 5** — Type safety
- **Vite 6** — Build tool & dev server
- **Tailwind CSS v4** — Utility-first styling
- **Motion (Framer Motion)** — Animations
- **Lucide React** — Icons
