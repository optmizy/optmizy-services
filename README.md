## 📸 Preview

A modern, A4-formatted services proposal page built with semantic HTML5, custom CSS animations, and Tailwind CSS utility classes. Features a dark color scheme with neon-green accents and smooth entry animations.

```
├── assets/
│   ├── css/

│   └── js/
│       └── tailwind.config.js  # Tailwind theme & color configuration
├── .gitignore
├── LICENSE
└── README.md
```

## 🚀 Getting Started

No build tools or package managers required. Simply open in a browser:

```bash
# Clone the repository
git clone https://github.com/optmizy/optmizy.git

# Navigate to project folder
cd optmizy

# Open in browser
open index.html
# or on Windows:
start index.html
```

> **Note:** An active internet connection is needed to load Google Fonts and the Tailwind CSS CDN.

## ✨ Features

- 🎨 **Dark Theme** — Deep navy/slate background with vibrant `#91D51F` green accents
- 📱 **Fully Responsive** — Mobile-first layout, scales from 320px to desktop
- 🖨️ **Print-Ready** — A4 print stylesheet for clean PDF exports
- ⚡ **Smooth Animations** — CSS keyframe animations with staggered delays
- 🎯 **Zero Dependencies** — No npm, no build step — just open and run
- ♿ **Accessible** — Semantic HTML5, proper heading hierarchy, SVG icons with `stroke`
